Elements

React implémente un système DOM indépendant du navigateur pour les performances et la compatibilité entre navigateurs. Ils appellent cela le DOM virtuel.

  • Un élément React n'est pas une instance réelle d'un [élément DOM] (https://developer.mozilla.org/en-US/docs/Web/API/Element)
  • React elements ont une API presque identique à DOM Elements

Les éléments React sont les blocs de construction des applications React. Un élément décrit ce que vous voulez voir à l'écran.

On pourrait confondre les éléments avec un concept plus connu de « composants ». En règle générale, les éléments ne sont pas utilisés directement, mais sont renvoyés par les composants.

Partie 1 - Hello World en JavaScript​

Dans cette partie du cours, nous allons créer une application "Hello World" en JavaScript vanille. Nous réécrirons plus tard l'application pour utiliser React.

  1. Créez le répertoire demos

    mkdir demos
    cd demos
    ```****

  2. Ouvrez demos dans l'éditeur de votre choix

    Si vous utilisez VS Code, vous pouvez taper la commande suivante sur la ligne de commande pour ouvrir l'éditeur

     code .

  3. Créez le fichier index.html

  4. Ajoutez le code suivant

    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <meta http-equiv="X-UA-Compatible" content="ie=edge" />
        <title>Document</title>
      </head>
      <body></body>
    </html>

Si votre éditeur prend en charge Emmet, vous pouvez utiliser l'extrait de code html:5

  1. Ajoutez un élément div avec un identifiant de root

    <body>
    +  <div id="root"></div>
    </body>

    If your editor supports Emmet you can use the code snippet div#root

  2. Add the following script tag to reference a JavaScript file that we will create in the next step.

    ```diff
    <body>
    <div id="root"></div>
    + <script type="text/javascript" src="/main.js"></script>
    </body>
    ```

    ! Assurez-vous que l'attribut src de la balise de script main.js commence par /, sinon les exemples de routeur ultérieurs ne fonctionneront pas correctement lorsque vous actualiserez la page.

  3. Créez un nouveau fichier demos/main.js

  4. Ajoutez le code suivant à demos/main.js

    const rootElement = document.getElementById('root');

    const element = document.createElement('div');
    element.textContent = 'Hello World';
    element.className = 'container';
    rootElement.appendChild(element);

  5. Créez un fichier package.json pour stocker les dépendances de votre projet local. Ouvrez une invite de commande/un terminal dans le répertoire demos et exécutez la commande suivante.

    npm init --yes

    --yes : tells npm says yes to accepting all the default values in the npm config file

  6. Installer un serveur Web

    npm install serve

En supposant que vous souhaitiez servir un site statique, une application d'une seule page ou simplement un fichier statique (que ce soit sur votre appareil ou sur le réseau local), serve est un serveur Web de développement qui sert du contenu statique.

 > Il se comporte exactement comme des déploiements statiques sur
 > https://vercel.com
 > il est donc parfait pour développer votre projet statique.

 > Pour plus d'informations voir :
 > https://www.npmjs.com/package/serve

  1. Configurez un "script npm" pour démarrer le serveur Web

    // package.json
    {
      "name": "demos",
      "version": "1.0.0",
      "description": "",
      "main": "index.js",
      "scripts": {
    +  "start": "serve -s -p 5000"
      },
      "keywords": [],
      "author": "",
      "license": "ISC",
      "dependencies": {
        "serve": "..."
      }
    }

  2. Exécutez le serveur Web

    npm start

  3. Ouvrez http://localhost:5000/ dans votre navigateur Chrome

14) Vérifiez que la page s'affiche :

```
Hello World
```

Partie 1 - Bonjour tout le monde en React​

Hello World en JavaScript n'est pas si différent de ce qu'il est en React. Mettons à jour le code pour le voir dans React.

  1. Ouvrez une nouvelle invite ou un nouveau terminal cmd (laissez serve en cours d'exécution) et installez React

    npm install react react-dom --save

  2. Ajoutez les balises de script pour inclure "Réagir" sur la page. Placez-les juste en dessous de la root div mais avant la balise main.js script

    <script src="/node_modules/react/umd/react.development.js"></script>
    <script src="/node_modules/react-dom/umd/react-dom.development.js"></script>

  3. Mettez à jour le code pour utiliser React

    const rootElement = document.getElementById('root');

    // const element = document.createElement('div');
    // element.textContent = 'Hello World';
    // element.className = 'container';
    // rootElement.appendChild(element);

    const element = React.createElement(
      'div',
      {
        className: 'container',
      },
      'Hello World'
    );

    ReactDOM.createRoot(rootElement).render(element);

La signature de la méthode pour createElement est la suivante :

React.createElement(type, [props], [...children]);

  1. Déconnectez-vous de React Element sur la console.

    const element = React.createElement(
      'div',
      {
        className: 'container'
      },
      'Hello World'
    );

    + console.log(element);
    ReactDOM.createRoot(rootElement).render(element);

  2. Ouvrez Chrome DevTools (F12 ou fn+F12 sur un ordinateur portable) pour voir la sortie de la console.

    Notez que l'élément React est juste un objet avec une propriété props qui contient un objet. Et l'objet props a trois propriétés que nous utilisons pour représenter l'élément HTML : type, children et className.

  3. Au lieu de comme troisième paramètre, les enfants (éléments enfants - même s'il ne s'agit que d'un élément de texte comme dans cet exemple) peuvent être passés dans le cadre des éléments props (qui est l'abréviation de propriétés).

    const element = React.createElement(
        'div',
        {
        className: 'container'
    +       children: 'Hello World'
    or
    +       children: ['Hello World', 'Goodbye World']
        },
    -     'Hello World'
    );

Pour résumer les paramètres de React.createElement sont les suivants.

  • Param 1 : l'élément que vous souhaitez créer
  • Param 2 : un objet qui contient toutes les propriétés que vous souhaitez appliquer ou définir sur cet élément
  • Param 3 : pour plus de commodité, vous pouvez fournir aux enfants n'importe quel nombre d'arguments après les props

React.createElement() effectue quelques vérifications pour vous aider à écrire du code sans bogue, mais il crée essentiellement un objet comme celui-ci :

// Note: this structure is simplified
const element = {
  type: 'div',
  props: {
    className: 'container',
    children: 'Hello World',
  },
};

Ces objets sont appelés "éléments React". Vous pouvez les considérer comme des descriptions de ce que vous voulez voir à l'écran. React lit ces objets et les utilise pour construire le DOM et le maintenir à jour.
