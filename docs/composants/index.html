Composants

Survol​

Les composants vous permettent de diviser l'interface utilisateur en pièces indépendantes et réutilisables et de penser à chaque pièce isolée.

Création d'un élément​

  1. Auparavant, nous avons créé un élément React et l'avons rendu comme indiqué ci-dessous.

    const element = <div className="container">Hello Element</div>;

    const rootElement = document.getElementById('root');
    ReactDOM.createRoot(rootElement).render(element);

Pour rendre cet élément réutilisable, nous devons le transformer en composant. Il existe deux types de composants: les fonctions et les classes dans React.

Création d'un composant de fonction​

Lorsque vous souhaitez réutiliser le code, vous l'extrairez souvent en une fonction.Les composants de la fonction de réaction vous permettent de réutiliser des bits d'interface utilisateur de la même manière.

  1. Créez la fonction suivante qui renvoie un élément JSX.

    function helloWorld() {
      return <div className="container">Hello Function Component</div>;
    }

Rendre un composant​

Les composants de la fonction et de la classe sont rendus de la même manière.

  1. Rendez le composant en utilisant le nom de classe ou de fonction comme nom de balise.Initialement, avoir la première lettre inférieure à inférieure.

    const root = ReactDOM.createRoot(document.getElementById('root'));
    root.render(<helloWorld />);

  2. Vous recevrez l'erreur suivante.

Composant et réutilisation​

Afin de réutiliser ce composant, nous avons besoin d'un autre composant d'emballage pour contenir plusieurs instances de notre composant.

  1. Créez un composant «App» et rendez plusieurs instances de «Helloworld» à l'intérieur.

    function HelloWorld() {
      return <div className="container">Hello Function Component</div>;
    }

    function App() {
      return (
        <div>
          <HelloWorld />
          <HelloWorld />
          <HelloWorld />
          <HelloWorld />
        </div>
      );
    }

    const root = ReactDOM.createRoot(document.getElementById('root'));
    root.render(<App />);

  2. Résultat

    Hello World
    Hello World
    Hello World
    Hello World

  3. Ouvrez «Chrome Devtools» et l'onglet «React Devtools» et montrez comment les composants se composent dans une application.

    <App>
      <div>
        <HelloWorld />
        <HelloWorld />
        <HelloWorld />
        <HelloWorld />
      </div>
    </App>

Création d'un composant de classe​

Alternativement, vous pouvez réutiliser le code en créant une classe, puis vous pouvez en créer plusieurs instances.La méthode «rendu» sur la classe est similaire à la fonction lorsque nous avons créé un composant de fonction.

  1. Créez une classe avec une fonction «render» qui renvoie `jsx».

    class HelloWorld extends React.Component {
      render() {
        return <div className="container">Hello Class Component</div>;
      }
    }

  2. Le texte s'affiche désormais comme indiqué ci-dessous.

    Hello Class Component
