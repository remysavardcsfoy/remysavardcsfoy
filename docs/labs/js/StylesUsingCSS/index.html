  • Implémentez vos CSS de Card du projet compsante-1

Implémentez vos CSS du projet compsante-1​

Collez le code CSS de votre projet composante-1 dans src/index.css afin qu'il remplace ce qui est actuellement là.

Enregistrez et revenez en arrière sur votre navigateur, et votre application devrait désormais avoir un style raisonnable.

À ce stade, notre application est un monolithe. nous devrons séparer en composants gérables et descriptifs. React n'a pas de règles précise pour ce qui est et n'est pas un composant - c'est à vous de le déterminer!

Les composants peuvent être stylisés à l'aide de classes CSS ou de styles en ligne comme vous le feriez lorsque vous utilisé HTML.

Implémentez vos CSS de Card du projet compsante-1​

className​

  • Utilisez className pas class
  • Passez une chaîne(string) à className
  1. Créez le fichier src/components/CardStyles.css
  2. Ajoutez vos style de Cards du projet composante-1

Cette configuration de projet utilise WebPack pour gérer tous les actifs. WebPack offre un moyen «prolonger» de faire des import (le concept d'importation) au-delà de JavaScript. Pour exprimer qu'un fichier JavaScript dépend d'un fichier CSS, vous devez importer le CSS à partir du fichier JavaScript:

CardStyles.css​

/* Vos styles de card */
.card {
    padding: 20px;
    border: 1px solid black;
    border-radius: 20px;
}

/* ... */

Card.js​

  import React from 'react';
+ import './CardStyles.css'; // Tell webpack that Button.js uses these styles

  export default function Card(props) {
    return (
      <div className="card">
        <h2>{props.title}</h2>
      </div>
    );
  }

En développement, l'expression des dépendances de cette manière permet à vos styles d'être rechargés à la volée lorsque vous les modifiés. En production, tous les fichiers CSS seront concaténés en un seul fichier .css minifié à la sortie du build.
