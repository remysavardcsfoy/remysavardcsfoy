Votre premier composant

Objectifs​

  • Effecuer le ménage
  • Créer un composant Card
  • Réutilisation et propriétés

Ménage préalable​

Create-React-App a généré quelques fichiers que nous n'utiliserons pas du tout pour notre projet.

  • Nous n'allons pas écrire les feuilles de style par composant, alors supprimez d'abord l'importation App.css depuis le haut du fichier App.js.
  • Nous n'utiliserons pas non plus le fichier logo.svg, alors supprimez cette importation également.

Dans VS Code, supprimez ces fichiers:

  • App.test.js
  • App.css
  • logo.svg
  • setupTests.js

Remarques:

  • Si vous avez arrêté votre serveur pour effectuer les tâches du terminal mentionnées ci-dessus, vous devrez le recommencer à l'aide de npm start.

JSX pour démarrer​

En tant que point de départ pour ce projet, nous allons fournir une fonction app () pour remplacer celle que vous avez maintenant.

Copiez l'extrait suivant dans votre presse-papiers, puis collez-le dans App.js afin qu'il remplace la fonction app () existante:

function App(props) {
  return (
    <main>
      <h1>Composante Card</h1>
      <div className="card">
         <h2>Titre de la Card</h2>
      </div>
    </main>
  );
}

Ouvrez public/index.html et modifiez le texte de l'élément <title> pour Composante Card. De cette manière, il correspondra au <h1> en haut de notre application.

<title>Composante Card</title>

Créer un composant <Card />​

Avant de pouvoir faire un composant, nous devons créer un nouveau fichier pour cela. En fait, nous devons faire un répertoire uniquement pour nos composants. À l'aide de VS Code ajoutez un répertoire nommé components à l'intérieur du répertoire src. Puis, dans ce répertoire, créez un fichier appelé Card.js. Assurez-vous que vous êtes à la racine de votre application avant de les exécuter!

src/components/Card.js

Notre nouveau fichier Card.js est actuellement vide! Ouvrez-le et écrivez la première ligne:

import React from "react";

Étant donné que nous allons faire un composant appelé Card, vous pouvez également commencer à ajouter le code pour Card.js, comme suit. Dans ce code, nous définissons la fonction et l'exportons sur la même ligne:

export default function Card() {
  return (
    // …
  );
}

C'est ok jusqu'à présent, mais notre composant doit retourner quelque chose! Revenez à src/App.js, copiez le code du div avec la class card et collez-la dans le fichier Card.js:

export default function Card() {
  return (
   <div className="card">
      <h2>Titre de la Card</h2>
   </div>
  );
}

Note: Les composants doivent toujours retourner quelque chose. Si à un moment donné dans le futur, vous essayez de rendre un composant qui ne retourne rien, React affichera une erreur dans votre navigateur.

Notre composant Card est terminé, du moins pour l'instant; Maintenant, nous pouvons l'utiliser dans App.js, ajoutez la ligne suivante près du haut du fichier pour importer Card:

import Card from "./components/Card";

Avec ce composant importé, vous pouvez remplacer l'élément <div> dans App.js par le composant <Card />.

  <main>
    <h1>Composante Card</h1>
-     <div className="card">
-       <h2>Titre de la Card</h2>
-     </div>
+     <Card />
  </main>

Faire un seul <Card />​

Les composants sont puissants car ils nous permettent de réutiliser des pièces de code et se référer à un seul endroit pour la gestion. Le problème est que nous ne voulons généralement pas réutiliser tout les éléments dans chaque composants; Nous voulons réutiliser la plupart des éléments et changer de petits morceaux. C'est là que les propriétés entrent en jeu.

Pluisuers <Card /> et la propriété title?​

Nous allons expérimenter avec la réutilisation. Vous allez devoir copier votre composants afin d'en avoirs plueieurs. Afin d'avoir plusieurs titres de Cards différents, nous devons nous assurer que chaque composant <Card /> effectue le rendu d'un title unique.

Dans App.js, ajoutez plusieurs Cards et donnez à chaque <Card /> un title:

  <Card title="Eat" />
  <Card title="Sleep" />
  <Card title="Repeat" />

Lorsque votre navigateur se rafraîchit, vous verrez les Cards avec le même titre qu'auparavant. Nous avons donné à notre <Card /> certaines propriétés, mais nous ne les utilisons pas encore. Revenons à Card.js.

Modifiez d'abord votre définition de la fonction Card () afin qu'elle prenne props comme paramètre.

Remplacez Titre de la Card par votre title. Lorsque vous êtes au milieu d'une expression JSX, vous utilisez des accolades pour injecter la valeur d'une variable.

En mettant tout cela ensemble, votre fonction Card() devrait se lire comme ceci:

export default function Card(props) {
  return (
   <div className="card">
      <h2>{props.title}</h2>
   </div>
  );
}

Votre navigateur devrait afficher trois Cards uniques.
