React

React est une bibliothèque JavaScript pour créer des interfaces utilisateurs

React (aussi appelé React.js ou ReactJS) est une bibliothèque JavaScript libre développée par Facebook (maintenant Meta) depuis 2013. Le but principal de cette bibliothèque est de faciliter la création d'application web monopage, via la création de composants dépendant d'un état et générant une page (ou portion) HTML à chaque changement d'état.

  • https://fr.wikipedia.org/wiki/React

Grâce à React, il est facile de créer des interfaces utilisateurs interactives. Définissez des vues simples pour chaque état de votre application, et lorsque vos données changeront, React mettra à jour, de façon optimale, juste les composants qui en auront besoin.

Créez des composants autonomes qui maintiennent leur propre état, puis assemblez-les pour créer des interfaces utilisateurs complexes.

React peut aussi être utilisé côté serveur avec Node, ou pour créer des applications mobiles grâce à React Native.

  • https://fr.reactjs.org/

Il existe tellement de bibliothèques et de frameworks JavaScript qu'il est difficile de suivre et de comprendre ceux qui valent la peine d'être appris et comment ils pourraient être utiles à votre organisation.

Facile à apprendre​

La bibliothèque React elle-même a un nombre de concepts très faible et est donc facile à apprendre.

Après avoir commencé à créer des applications avec la bibliothèque React, vous apprenez rapidement qu'elle fait une chose et qu'elle le fait bien, mais vous avez besoin de beaucoup d'autres choses pour créer une application Web en l'utilisant.

Écosytème​

React dispose de tout un écosystème d'autres outils et bibliothèques pour combler ces lacunes. Considérez la création d'une application React comme étant similaire à l'achat d'un logiciel de pointe et à son intégration. Vous intégrez React à d'autres bibliothèques et outils de pointe pour créer une application. Voici une liste rapide de certaines des bibliothèques et outils les plus populaires souvent utilisés avec React.

  • Babel : un compilateur JavaScript qui vous permet d'utiliser les dernières fonctionnalités du langage JavaScript dès aujourd'hui
  • Webpack : regroupe efficacement votre JavaScript, CSS, HTML et vos images pour le déploiement
  • Application Create-React : crée un projet React avec les meilleures pratiques en exécutant une seule commande
  • TypeScript : TypeScript est un sur-ensemble typé de JavaScript qui se compile en JavaScript brut. Le compilateur Babel ou TypeScript est utilisé pour obtenir les fonctionnalités d'un langage de programmation moderne aujourd'hui
  • Next.js : les fonctionnalités dont vous avez besoin pour la production : rendu hybride statique et serveur, prise en charge de TypeScript, regroupement intelligent, préchargement de route, etc. Aucune configuration nécessaire.
  • Bibliothèques de gestion d'état, y compris Redux et MobX pour concevoir et gérer les données de votre application
  • GraphQL : un langage de requête pour vos API de données
  • React Native : créez des applications natives pour Android et iOS à l'aide de React
